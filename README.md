# GAME CONTROLS:
 
WASD - Move

Left Shift - Run

Space - Jump

E - Toggle Build Mode

# BUILD CONTROLS:

Scroll Wheel/Button Menu - Switch through objects

Left Click - Spawn Object

Right Click - Delete Object

E - Toggle Build Mode
